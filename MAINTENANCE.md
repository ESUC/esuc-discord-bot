# Maintenance Guide

## Setup
Create .env file with DISCORD_TOKEN='<Your Token>' found from discord 
developer token. DO NOT SHARE WITH PUBLIC.

In main repo directory run `pip3 install -r requirements.txt `(pip on 
windows).

Invite bot to development server via discord developers OAuth2 
generated URL.

Note: Bot is smaller than needed for sharding, do not implement.
